# Data-Analytics-Dashboard – Spotify Chart Insights
## Project Objective
To analyze Spotify chart data and identify key factors that influence a song’s popularity and ranking. The project uncovers trends, patterns, and correlations through data cleaning, visualization, and insight generation to understand music performance on streaming platforms.

## Dataset used
<a href="https://github.com/monishv080603-hue/Data-Analysis-Dashboard/blob/main/spotify-top-50-world.csv">Dataset</a>

## Key Performance Indicators (KPIs) Used
- Distinct Songs Count – Total number of unique songs in the dataset.
- Count of Artists – Total number of unique artists appearing in the charts.
- Average Song Duration – Measures typical track length and duration trends.
- Average Popularity Score – Overall popularity of songs across the dataset.
- Songs by Artist – Number of songs contributed by each artist.
- Popularity Distribution – Tracks how popularity scores vary across songs.
- Explicit vs Non-Explicit Songs – Comparison of counts and popularity performance.
- Songs by Release Year – Identifies yearly music trends and active release periods.
- Average Popularity by Album Type – Measures how singles, albums, and compilations perform.
- Average Popularity by Month & Quarter – Shows seasonal or quarterly popularity trends.

## Project Process
- Collected the Data
Loaded the Spotify chart dataset with details like songs, artists, popularity, album type, and release dates.
- Cleaned the Data
Removed duplicates, handled missing values, formatted dates, and made sure all columns were ready for analysis.
- Explored the Data
Looked into how popularity, duration, artists, and release years were distributed to understand the overall structure.
- Created KPIs
Built key metrics such as distinct songs, artist count, average duration, average popularity, explicit vs non-explicit comparison, songs by year, and popularity by album type, month, and quarter.
- Visualized the Data
Designed charts to see trends in popularity, artist activity, release patterns, and explicit vs non-explicit performance.
